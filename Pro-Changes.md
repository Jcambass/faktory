## 1.3.0

* Validate statsd tags, per Datadog docs [#283]
* Add periodic job history and details page [#265]
* Allow cron jobs to specify static arguments:
```toml
[[cron]]
  schedule = "* * * * *"
  [cron.job]
    type = "SomeWorker"
    args = ["minutely"]

[[cron]]
  schedule = "0 * * * *"
  [cron.job]
    type = "SomeWorker"
    args = ["hourly"]
```
